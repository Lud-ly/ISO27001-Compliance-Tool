# ISO27001-Compliance-Tool
Outil Python automatisé pour évaluer la conformité ISO 27001 avec génération de rapports PDF

Fonctionnalités :
- Checklist interactive des 93 contrôles ISO 27001:2022

- Scoring automatique par domaine (A.5 à A.8)
Génération de rapports PDF avec graphiques (compliance rate, gaps analysis)

- Export JSON/CSV pour suivi dans le temps
Dashboard web avec Flask pour visualisation
